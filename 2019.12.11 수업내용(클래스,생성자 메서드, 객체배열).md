### 생성자 메서드(constructor)

- 클래스를 객체 생성할 때 호출되는 메서드.

  new 클래스명( )

  -------------> 생성자 메서드(소괄호)

  ex) Data d = new Data( );

- 모든 클래스는 1개 이상의 생성자 메서드를 가지고 있어야 한다.

- 클래스의 소스에 생성자 메서드가 정의되어 있지 않다면 컴파일러가 생성자를 만들어 준다. => 디폴트 생성자

- 생성자가 없는 경우에도 디폴트 생성자 하나가 생성된다.

- 생성자 메서드 정의 방법

  - 메서드 명은 클래스 명과 동일해야 한다.
  - 매개변수는 선택적이다. (여러개 오버로딩 가능)
  - 리턴값의 타입은 생략한다.
  - 객체 생성시 수행해야 하는 기능 또는 객체 생성시 데이터를 전달받아 멤버변수들의 값을 초기화하는 기능



- import : 같은 패키지가 아닌 다른 패키지에 있는 데이터를 접근하기 위해 쓰는 용어

#### this

- this란 java가 내장하고 있는 리터럴. - 자기 자신을 참조한다.

  - 자기 자신에 대한 객체값을 갖고 있다.

  this.xxx : 나의 멤버 xxx와 같은 의미(뒤가 변수이던 메서드이던)

  this( ) => 메서드 호출 용도. 생성자 메서드 내에서만 호출 가능

  ​	생성자 메서드의 첫 행에서만 호출이 가능하다.

  ​	동일한 클래스 내에 있는 다른 생성자를 호출
